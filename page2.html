我的學習紀錄

我主要是透過 YouTube 影片、網路資料和自己實際修改檔案來學習。下面是我這次做網站的大致過程：

第一階段：先看懂 HTML 和 CSS

一開始我是直接看影片學，先認識標題、段落、圖片、連結這些基本標籤，再慢慢理解 CSS 可以控制顏色、大小和位置。影片原本是教做一個介紹店家的網頁，我把它改成自己的自主學習網站。

第二階段：把範例改成自己的內容

我把原本範例的文字、圖片和連結換成自己的資料。這個階段最常遇到的問題是排版跑掉，所以我會一邊改 CSS，一邊重新打開網頁看效果。

第三階段：開始練習 JavaScript

JavaScript 對我來說比較難，因為它不像 HTML 和 CSS 那麼直覺。我先跟著影片做代辦事項清單網頁，了解按鈕、輸入框和事件觸發的概念。

第四階段：整理網站和檢查手機版

最後我把頁面內容整理成比較容易閱讀的樣子，也試著讓手機版不要跑版。這次作品還有很多可以改進的地方，但我比較知道網站是怎麼一步一步做出來的。

第五階段：了解 Codex 的運用方式

後來我也開始了解 Codex 可以怎麼協助寫程式。我不是直接重做整個網站，而是先保留原本架構，再請它幫忙檢查哪裡排版不太穩、哪裡標籤有錯，並協助調整顏色、間距、動畫和手機版顯示。這個網站後面的美化和修正，大部分都是靠 Codex 幫我一起完成的，我也從過程中學到要怎麼描述問題、看修改結果，再自己判斷要不要保留。
